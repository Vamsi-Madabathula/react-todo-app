React To-Do List App
A To-Do List application built using React. This project allows users to add and delete tasks dynamically.

🚀 Features

Delete tasks
Dynamic task rendering using React
Simple and clean UI
Component-based architecture

📌 How It Works
site is live at https://vamsi-madabathula.github.io/react-todo-app/

Users enter a task in the input field.
Clicking the Add button stores the task in a state array.
The task list is rendered using the map() function.
Clicking Delete removes the task from the array using filter().
